About this Website

Browser compatbility

This website requires a modern browser that supports HTML5 + CSS 3 + ECMAScript 6 (ES6). Chrome is recommended.

No attempt is made to support downlevel browsers; because doing so would add complexity, and come at the expense of those using modern browsers.

Tools used

This (static) website is powered by GitHub Pages; which is free, free of ads, and has no strings attached (no attempts to upsell you).

It was coded using (free) Visual Studio Code.

The Github Repository for this project is public.
